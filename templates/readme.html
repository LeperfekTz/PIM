🧠 ChatBot Web com Flask, MongoDB e API Local

Este é um projeto de chatbot com autenticação de usuários, integração com uma API local e armazenamento de conversas usando MongoDB. O site foi construído com Flask e oferece funcionalidades como login, registro, redefinição de senha e histórico de mensagens por usuário.

🚀 Funcionalidades

  • ✅ Login e registro de usuários
  • 🔐 Redefinição de senha via e-mail
  • 💬 Chat com API local via POST
  • 💾 Salvamento de mensagens e respostas no MongoDB
  • 📜 Histórico de conversas por usuário
  • 👤 Sessão para manter usuário logado

🛠️ Tecnologias Utilizadas

  • Python 3
  • Flask
  • MongoDB (Atlas ou local)
  • Flask-Mail (para envio de e-mails)
  • itsdangerous (tokens seguros)
  • dotenv (configurações)
  • HTML/CSS (Jinja2)
